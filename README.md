# 📚 Sistema de Gerenciamento de Biblioteca

Projeto desenvolvido durante o 1º semestre do curso de Ciências da Computação.
Sistema em Python para gerenciar o acervo de uma biblioteca, com funcionalidades
de cadastro, busca e controle de empréstimos de livros.

---

## 🚀 Funcionalidades

- Cadastrar novos livros no acervo
- Listar todos os livros disponíveis
- Buscar livros por título ou autor
- Registrar empréstimos e devoluções

---

## 🛠️ Tecnologias utilizadas

- Python 3

---

## ▶️ Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/DESTRUIDOR2029/sistema-biblioteca
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd sistema-biblioteca
   ```
3. Execute o programa:
   ```bash
   python main.py
   ```

---

## 📋 Requisitos

- Python 3.8 ou superior
